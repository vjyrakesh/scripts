***Git command shortcut scripts
```
ga.bat     == git add .
gb.bat     == git branch
gc.bat     == git commit -m [message]
gcb.bat    == git checkout [branch] 
gcnb.bat   == git checkout -b [branch]
gpob.bat   == git push origin [branch]
gpom.bat   == git push origin master
gs.bat     == git status
```
